# Payvex — Intelligent Payment Solutions

A modern, single-page marketing website for a payment gateway platform.
Built as a fully static site.

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Fraunces, DM Sans)
- AVIF images for optimised asset delivery

## Features

- Animated hero section with gradient blobs and live dashboard mockup
- Responsive navigation with mobile slide-in drawer
- Animated stat counters (hero and metrics band)
- Features grid with image cards and hover effects
- How It Works section with step images and API code preview
- Solutions tabbed panel (Fintechs, Enterprises, Platforms, Banks)
- Testimonials, CTA banner, and multi-column footer
- Scroll reveal animations on all major sections
- Image skeleton shimmer loader with fade-in on load
- Fully responsive at 900px and 480px breakpoints
